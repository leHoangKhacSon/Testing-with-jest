# Testing API with Jestjs

## Installation

using npm:
```
$ npm install
```

using yarn:
```
$ yarn install
```
## Setting

Step 1: Create .env file in server folder 
Step 2: Create key PORT={value} and key SECRECT_KEY={random value}
Step 3: Completed

## Running test

using npm:
```
$ npm test
```

using yarn:
```
$ yarn test 
```
